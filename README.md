# Grocery WebApp

A full-stack grocery web application built with MongoDB, Express, React, and Node (MERN stack).

## Features
- Browse and search for products
- User-friendly shopping interface

## Technologies Used
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB

## Getting Started

### Prerequisites
- Node.js, npm
- MongoDB
